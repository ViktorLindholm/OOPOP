A state full FIFO queue process.

Authors: Karl Marklund (karl.marklund@it.uu.se).

Description

A state full FIFO queue process. A common pattern in Erlang is to provide services as separate processes. If we keep the FIFO queue as a separate process, we can use the PID to the FIFO process as a reference to the FIFO. When we update the state of the FIFO, the PID remains the same.
